FOSSTorio
=========

FOSSTorio is an open-source re-implementation of [Factorio](https://www.factorio.com/).
Currently in **EXTREMELY** early stage of development, and not usable to any extend.

A word of warning
-----------------

This software isn't, in any way shape or form associated with Wube Software.
You will need resources from original game, I strongly discourage any attempts to acquire them in any way, except copying from your own copy of the game.

Resources
---------
[Factorio LUA API](https://lua-api.factorio.com/)

Global goals
-----

* Implement basic functionality to be considered "playable".
* Reach feature parity with Factorio.
* Optimize, to run as good as Factorio or better.
* Implement saves compatibility with Factorio.
* Implement mobile version, with mobile friendly UX.
